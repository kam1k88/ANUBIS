# ANUBIS (В разработке)

Робот-аналитик, который видит судебную систему России целиком. Не судит - но знает, как судят. Не выносит приговоры - но показывает, чем дело закончится с высокой точностью. Не заменяет юристов - но делает их сильнее. ANUBIS умеет: разбирать любое дело - от уголовки до арбитража, находить слабые места в позиции сторон, прогнозировать возможные решения, анализировать стиль каждого судьи, прокурора и адвоката, сопоставлять законы с реальной практикой, собирать доказательства и документы, строить графы судебных связей по всей стране. ANUBIS - это рентген судебной системы. Первый инструмент, который показывает, как работает право в реальности.
